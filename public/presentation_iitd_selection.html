## Introduction and Goals ## * Gene3d is a fork of Gene: Gyrokinetic plasma microturbulence simulator * Real coordinates: $ x, y, z, v_{||}, \mu, n $ * Prohibitive memory requirements: - strongly shaped flux surfaces - high gradients near the edge of the plasma - grid resolution is therefore important * Switching from double to single precision gave acceptable results * ***Reduce memory foot print—improve performance*** - by attempting lossy compression of data - by modifications that reduce memory - to conclude on feasibility of compressors

Plan

  1. Find what data is compressible in Gene3d
  2. Check if memory footprint can be reduced by other means
  3. Implement compression tests to check accuracy
  4. Implement compression in a way that actually reduces the memory footprint
